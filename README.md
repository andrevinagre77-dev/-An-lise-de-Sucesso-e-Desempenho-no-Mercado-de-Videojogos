# -An-lise-de-Sucesso-e-Desempenho-no-Mercado-de-Videojogos
Este projeto realiza uma análise exploratória e estatística exaustiva do mercado de videojogos, utilizando dados históricos até 2016. O objetivo principal é identificar padrões que determinam o sucesso de um jogo (vendas, críticas e plataformas) para prever campanhas publicitárias e tendências para o ano de 2017.
