# -An-lise-de-Sucesso-e-Desempenho-no-Mercado-de-Videojogos
Este projeto realiza uma análise exploratória e estatística exaustiva do mercado de videojogos, utilizando dados históricos até 2016. O objetivo principal é identificar padrões que determinam o sucesso de um jogo (vendas, críticas e plataformas) para prever campanhas publicitárias e tendências para o ano de 2017.

A análise segue um rigoroso pipeline de ciência de dados, desde a limpeza técnica até à validação de hipóteses estatísticas, permitindo uma tomada de decisão baseada em evidências para o planeamento de inventário e marketing.

🚀 Pipeline de Análise
O projeto foi executado seguindo os seguintes passos fundamentais:

Auditoria e Limpeza de Dados: Tratamento de valores ausentes (como o caso "TBD" em avaliações) e conversão de tipos de dados para análise numérica.

Análise Exploratória (EDA): Mapeamento do ciclo de vida das plataformas e identificação das consolas líderes de mercado.

Análise de Correlação: Estudo do impacto das avaliações de críticos e utilizadores nas vendas globais.

Perfil Regional: Comparação de preferências de géneros e plataformas entre as regiões NA, EU e JP.

Testes de Hipóteses: Aplicação de testes T de amostras independentes para comparar médias de classificações entre plataformas e géneros.

💡 Principais Insights
Ciclo de Vida das Plataformas: Identificou-se que as plataformas levam cerca de 5 anos a atingir o pico de vendas e têm uma vida útil média de 10 anos antes do declínio total.

Drivers de Negócio: As notas dos críticos têm uma correlação positiva mais forte com o sucesso financeiro do que as notas dos utilizadores.

Diferenças Culturais: Enquanto o género "Action" e plataformas como PS4 dominam o Ocidente, o mercado japonês apresenta uma preferência distinta por dispositivos portáteis e géneros de RPG.

🛠️ Tecnologias Utilizadas
Linguagem: Python.

Bibliotecas Principais:

Pandas para manipulação e estruturação de dados.

Matplotlib e Seaborn para visualizações estatísticas e de tendências.

Scipy.stats para a realização de testes estatísticos rigorosos.

📈 Conclusões para 2017
O projeto culmina na recomendação de focar investimentos em plataformas em ascensão (como PS4 e Xbox One) e géneros com maior estabilidade de vendas, ajustando a estratégia de acordo com as particularidades regionais detetadas na análise.
